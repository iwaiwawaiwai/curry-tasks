毎朝のチェックインを実行してください。

## 実行フロー

1. **brain/00_Inbox/ と 20_Projects/ を確認**して、未完了・進行中のタスクを収集する
2. **今日の役割（帽子）を1つ提示する**
   - 経営者 / PM / エンジニア のどれか
   - 前日や直近のノートから文脈を読んで推奨する
3. **今日やること（最大3つ）を提示する**
   - 緊急度・重要度・文脈（打ち合わせ予定など）を考慮する
   - 多く出さない。3つで十分
4. **チェックイン結果を brain/10_Journal/ に保存する**
   - フォーマット: `YYYY-MM-DD_check-in.md`

## 出力形式

```
# 今日のチェックイン — YYYY-MM-DD

## 今日の帽子
{経営者 / PM / エンジニア} — {一言理由}

## 今日やること（3つ）
1. {タスク}
2. {タスク}
3. {タスク}

## 一言
{今日の自分への短いメッセージ}
```

## ルール
- タスクは「完了できるサイズ」に分解する
- 3つを超えて提示しない
- 気分の記録は任意（ユーザーが話してくれたら記録する、聞かない）
