# メルカリ出品画像フォルダ

## フォルダ命名規則

```
{4桁ID}/
```

例：
```
mercari-items/
├── 0001/
│   ├── 01.jpg
│   └── 02.jpg
├── 0002/
│   └── 01.jpg
```

## TSVカラム構成（6列）

| # | カラム | 内容 |
|---|--------|------|
| 1 | ID | 4桁数字（フォルダ名と一致） |
| 2 | タイトル | AI生成、36文字以内 |
| 3 | カテゴリー | メルカリ正式名称 |
| 4 | 商品の状態 | メルカリ定型文 |
| 5 | 販売価格 | 半角数字 |
| 6 | 商品説明文 | エピソード＋スペック＋ハッシュタグ（1000字以内） |

出力タイトル（メルカリ入力用）= `{ID} {タイトル}` （最大41文字）

## 使い方

1. `{ID}/` フォルダを作成して画像を入れる
2. Claude Code にID（またはフォルダ）と商品情報を伝える
3. 6列TSVを受け取り、スプレッドシートに貼り付ける
4. ブックマークレットで自動入力
