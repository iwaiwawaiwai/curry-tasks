KAORILOGO AIコンシェルジュ｜データスキーマ定義書

作成日: 2026-03-22　／　最終更新: 2026-03-23　／　対象: プロトタイプ（Node.js版）実装ベース
情報源： サイト情報 公式サイト・ECから取得 こちらで付与 AIまたは手動アサイン サイト+加工 サイト情報を整理・要約
｜
DB制約： PK 主キー FK 外部キー NOT NULL 必須 UNIQUE 一意 NULL可 省略可 ENUM 選択肢固定

① 商品マスタ（products）

各アロマ商品の基本情報。プロトタイプ対象: 13件（Life of Aromaブレンド5件・エッセンシャルオイル5件・KAORILOGOオリジナル3件）。参照: kaorilogo-products.json および server.js 商品データセクション

フィールド名 日本語名 データ型 制約 情報源 備考
id 商品ID VARCHAR(50) PKNOT NULLUNIQUE こちらで付与 管理用ユニークID（例: eo_lavender, blend_on_time）
brand ブランド名 ENUM NOT NULLENUM サイト情報 'Life of Aroma' | 'KAORILOGO'
name 商品名（英語） VARCHAR(100) NOT NULL サイト情報
name_ja 商品名（日本語） VARCHAR(100) NULL可 サイト情報 英語名のみの商品は省略可
category カテゴリ ENUM NOT NULLENUM サイト+加工 essential_oil | blend_aroma | travel_aroma | business_card_aroma | original_blend
description 説明文 TEXT NULL可 サイト+加工 商品ページの説明文を要約・整形
components 成分・原料 JSON (array) NULL可 サイト情報 精油名の文字列リスト。ブレンド品の比率は未取得
effects 効果・効能タグ JSON (array) NULL可 サイト+加工 AIの推薦文生成に使用。説明文から抽出・整理
url 商品ページURL TEXT NULL可 サイト情報 推薦結果の「詳細を見る」リンクに使用
formats 販売形態・容量 JSON (array) NULL可 サイト情報 例: ["5ml", "10ml", "ロールオン"]
scenes 推奨シーン JSON (array) NULL可 こちらで付与 AI推薦ロジックに使用。訪問後に要確認・修正

② 6軸プロファイル（profile）

AIが会話から自動的に推定・更新するユーザープロファイル。スコアはAIが内部でアサイン。現行実装: INT (-2〜+2) または文字列・null。すべてこちらで付与。

フィールド名 日本語名 データ型 制約 情報源 備考（軸の両端）
profile.activation 活性度 INT (-2〜+2) NULL可 こちらで付与 -2=深い鎮静・睡眠　↔　+2=高い活性・集中。未判定時はnull
profile.sensory 感覚傾向 INT (-2〜+2) NULL可 こちらで付与 -2=クール・爽やか・シャープ　↔　+2=ウォーム・甘み・まろやか
profile.social 社交性 INT (-2〜+2) NULL可 こちらで付与 -2=プライベート・内向き　↔　+2=パブリック・コミュニケーション
profile.formality フォーマル度 INT (-2〜+2) NULL可 こちらで付与 -2=カジュアル・現代的　↔　+2=フォーマル・伝統・信頼感
profile.scene 使用シーン STRING | null NULL可ENUM的 こちらで付与 'work' | 'bedroom' | 'living' | 'gift' | 'travel' など。未判定時はnull
profile.time 使用時間帯 STRING | null NULL可ENUM的 こちらで付与 'morning' | 'daytime' | 'evening' | 'night' | 'anytime' など。未判定時はnull

③ AIレスポンスJSONスキーマ（/api/chat レスポンス）

Claude APIから返却されるJSONの構造定義。server.jsでパースしてフロントエンドに渡す。

フィールド名 日本語名 データ型 制約 備考
reply AIの返答テキスト STRING NOT NULL チャット画面に表示するテキスト
profile 6軸プロファイル OBJECT NOT NULL ② の6軸を含むオブジェクト。各軸はnullまたはスコア値
filled_axes 判定済み軸数 INT (0〜6) NOT NULL 4以上でhearingからproposalモードへ自動移行
current_agent 現在のモード ENUM NOT NULLENUM 'hearing' | 'proposal' | 'closing'　の3モード自律切替
recommendations 推薦商品リスト ARRAY (max 5件) NOT NULL 全13商品を内部スコアリングし上位5件のみ返却
recommendations[].rank 順位 INT (1〜5) NOT NULL
recommendations[].product 商品名 STRING NOT NULL ① 商品マスタの name と対応
recommendations[].match_score マッチスコア INT (0〜100) NOT NULL プロファイルとの一致度。フロントの進捗バーに表示
recommendations[].reason 推薦理由 STRING NOT NULL 1行の推薦理由テキスト。推薦パネルに表示

④ 今後追加予定のフィールド

本番実装・BASEサイト連携時に追加が必要な項目。

フィールド名 日本語名 データ型 制約 情報源 優先度・備考
price 販売価格（税込） INT NULL可 サイト情報 🔴 高：推薦カードに価格表示が必要。BASEサイトで確認要
image_url 商品画像URL TEXT NULL可 サイト情報 🔴 高：推薦カードのビジュアルに必須
base_url BASEショップURL TEXT NULL可 サイト情報 🔴 高：推薦結果から直接購入ページへの遷移に使用（lifeofaroma.thebase.in）
stock_status 在庫状況 ENUM NULL可ENUM サイト情報 🟡 中：'in_stock' | 'out_of_stock' | 'discontinued'
blend_ratio ブレンド比率（%） JSON (object) NULL可 ヒアリング取得 🟡 中：鶴香・瀬戸の香のaxesスコア精度向上に必要
