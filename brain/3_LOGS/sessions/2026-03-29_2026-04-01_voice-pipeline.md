## 決めたこと・なぜそう決めたか

| 決定 | 理由 |
|---|---|
| WhisperX → MLX Whisper に変更 | WhisperX は速度が致命的（60秒に数分）。MLX Whisper は Apple Silicon 最適化で超高速 |
| 話者分離は不要 | 目的は議事録の「目的・結論・アクションアイテム」の抽出。誰が言ったかは不要 |
| Anthropic API は使わない | 音声→テキスト変換はローカル完結。テキスト→タスク化は Claude Code に手渡す運用で十分 |
