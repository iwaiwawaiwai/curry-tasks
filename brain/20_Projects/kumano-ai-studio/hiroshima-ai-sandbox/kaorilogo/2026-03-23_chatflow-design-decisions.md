## 確定した設計方針

### 1. リアルタイム推薦パネル — **採用**

画面を2カラムに分割する。

```
┌──────────────────────┬──────────────────────────┐
│  💬 チャット          │  🌸 あなたの香りのプロファイル│
│                      │                          │
│  AI: どんな状況ですか？│  気分:   ■□□□□  鎮静    │
│  User: 仕事中で…     │  感触:   □□■□□  ニュートラル│
│  AI: 集中したい感じ？ │  シーン: 寝室             │
│  User: いや、落ち着   │  時間帯: 夜               │
│       きたいんです    │   ↑ 会話に応じて埋まっていく│
│                      │                          │
│                      │ ────────────────────── │
│                      │                          │
│                      │  ✨ おすすめの香り         │
│                      │  ① OFF Time              │
│                      │  ② ラベンダー             │
│                      │  ③ ベルガモット           │
│                      │   ↑ 常時表示・会話で更新  │
└──────────────────────┴──────────────────────────┘
```

**右パネルの構成（上下2段）:**
- **上段 — 香りのプロファイル（表示のみ・編集不可）:**
  - 6軸スコアが会話とともに埋まっていく。「わかってもらえている」感を可視化
  - **会話の要約として機能する** → ユーザーが見て「ちょっと違う」と思ったら、チャットで「もう少し甘い感じが好き」などと伝える
  - プロファイルの更新はAIのみが行う。ユーザーは会話を通じてのみ影響を与えられる
- **下段 — おすすめの香り:** 初回から暫定表示。会話が進むほど精度が上がる。機能志向ユーザーはここから直接選べる

**採用理由:**
- 上段でプロセスが見える → 会話型ユーザーの安心感・信頼感
- 下段で常に選択肢がある → 機能志向ユーザーを取りこぼさない
- 後から追加するとバックエンドから作り直しになるため先行決定

---

### 2. 3エージェント構成 — **自律切替**

| エージェント | 役割 |
|---|---|
| ヒアリングAgent | 気分・目的・シーンを会話で深掘り |
| 提案Agent | 6軸スコアを元に商品をレコメンド |
| クロージングAgent | 購入へのナビゲーション |

**切替ロジック:**
```
ヒアリング → 軸スコアが一定以上揃った → 提案へ自動移行
提案中     → 「違う」「やっぱり」を検知 → ヒアリングへ戻る
           → 「これにします」を検知   → クロージングへ
```

- 確認プロンプト（「切り替えていいですか？」）は**撤廃**
- フローはウォーターフォールではなく**双方向・非線形**

---

### 3. 提案時の確認 — **撤廃**

- 「提案してもいいですか？」は出さない
- 会話中から常に現時点の候補を右パネルに表示
- ユーザーが望めば深掘り、すぐ買いたければ右パネルから選べる

### 4. リアルタイム推薦の開始タイミング — **初回発言から**

```
発言0（会話開始）: 人気商品を暫定表示 → ALL Time, OFF Time, ラベンダー等
発言1（シーン）:   シーンで絞り込んだ候補に更新
発言2（時間帯）:   さらに絞り込まれる
発言3〜（気分）:   プロファイル一致度が上がり精度向上
```

**体験として：** 話すたびに右パネルが変化することでユーザーが「AIが自分を理解していっている」と感じる

## 対応するユーザー類型

| タイプ | 行動 | このUIでの体験 |
|---|---|---|
| 会話型 | じっくり話して決めたい | チャットを深めながら推薦が精度向上 |
| 機能型 | 手っ取り早く欲しい | 右パネルの候補を最初から見て選べる |
